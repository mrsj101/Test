# Design Tokens

This project uses a small set of design tokens to keep styles consistent.

## Spacing Scale

| Token | Value |
|-------|-------|
| `space-1` | `0.25rem` |
| `space-2` | `0.5rem` |
| `space-4` | `1rem` |
| `space-6` | `1.5rem` |
| `space-12` | `3rem` |

## Font Sizes

| Token | Value |
|-------|-------|
| `font-base` | `1rem` |
| `font-lg` | `1.2rem` |
| `font-xl` | `clamp(1.5rem, 4vw, 2.5rem)` |

## Color Palette

| Token | Value |
|-------|-------|
| `--bg-color` | `#0E0E10` |
| `--bg-color-alt` | `#121212` |
| `--text-color` | `#FFFFFF` |
| `--text-color-muted` | `rgba(255, 255, 255, 0.9)` |
| `--accent-color` | `#C9A857` |
